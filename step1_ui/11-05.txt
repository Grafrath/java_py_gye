js에서 취급하는 데이터 타입

number : 정수 실수
string : 문자
bool : 참, 거짓을 데이터로
undefined : 값이 할당되지 않음
null : 의도적으로 비워둠
bigint : number보다 큰 숫자 좀 더 정교한 연산 가능(뒤에n)

변수
데이터를 담을수 있는 이름있는 그릇
변수명을 통해 안에 있는 데이터를 식별할 수 있다
var : 같은 이름의 변수가 만들어졌을 때를 감지하지 못한다.
let : 같은 이름의 변수가 만들어졌을 때를 감지한다 (mutable)
const : 일종의 상수 (immutable) 값 재대입 불가

초기화
변수에 값을 할당

let은 초기화 가능 const는 초기화 불가능
